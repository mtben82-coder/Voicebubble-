# Background Images

This folder should contain background images for notes.

## Required Images:

Place PNG files with these names:
- `mountain.png` - Mountain landscape
- `ocean.png` - Ocean/water scene
- `forest.png` - Forest/trees
- `city.png` - City skyline
- `space.png` - Space/stars
- `waves.png` - Abstract waves
- `circles.png` - Abstract circles
- `geometric.png` - Geometric patterns

## Image Specifications:
- Format: PNG with transparency (recommended)
- Size: 1920x1080 or similar HD resolution
- Keep file sizes reasonable (under 500KB each)
- Images will be displayed with 15% opacity for readability

## Note:
If images are not present, the app will fall back to gradient backgrounds automatically.
